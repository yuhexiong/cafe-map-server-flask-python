# Cafe Map Server


### Data From [Cafenomad](https://cafenomad.tw/)
整合 Cafenomad API，並使用 OpenStreetMap 在伺服器上顯示資訊。

## Overview

- **程式語言**：Python v3.9.15  
- **Web 框架**：Flask v3.0.0  

## Run

```bash
python app.py
```
