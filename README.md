# Cafe Map Server

### DataSet From [Cafenomad](https://cafenomad.tw/)
Integrate cafenomad api and use openstreetmap to show info on server

## Overview

- Language: Python v3.9.15
- Web FrameWork: Flask v3.0.0

### run

```
python app.py
```
